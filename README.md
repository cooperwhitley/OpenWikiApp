### User Stories

#### As a user I want to...

- View articles without signing in
- Sign up for an account
- Log in/out
- Write an article, which includes...
  - A title
  - A category
  - A summary paragraph
  - Image upload
  - A quick information box
  - Sub sections
  - References
  - Links to other articles
- Discuss an article on its own discussion page
- Edit any article
